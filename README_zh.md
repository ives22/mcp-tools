## 什么是 MCP？

**模型上下文协议 (MCP)** 是一个开放标准，使 AI 模型和智能体能够安全地连接外部数据源和工具。可以把它理解为 AI 的通用"USB-C 接口"——一旦工具支持 MCP，任何兼容 MCP 的 AI 都可以直接使用，无需定制集成。

## 📚 仓库结构

| 目录 | 描述 |
|------|------|
| `servers/` | MCP 服务器实现 |
| `tools/`   | 独立 MCP 工具和实用程序 |
| `examples/`| 使用示例和配置文件 |
| `docs/`    | 文档和使用指南 |

## 🚀 快速开始

```bash
# 克隆此仓库
git clone https://github.com/ives22/mcp-tools.git
cd mcp-tools

# 查看可用工具
ls servers/
ls tools/
```

## 📦 添加工具

1. 在对应目录下创建你的工具（`servers/` 或 `tools/`）
2. 包含 `README.md` 说明安装和使用方法
3. 如适用，提供 `mcp.json` 配置文件
4. 提交 Pull Request
