## What is MCP?

The **Model Context Protocol (MCP)** is an open standard that enables AI models and agents to securely connect with external data sources and tools. Think of it as a universal "USB-C port" for AI — once a tool supports MCP, any MCP-compatible AI can use it without custom integrations.

## 📚 Repository Structure

| Directory | Description |
|-----------|-------------|
| `servers/` | MCP server implementations |
| `tools/`   | Standalone MCP tools and utilities |
| `examples/`| Usage examples and configurations |
| `docs/`    | Documentation and guides |

## 🚀 Quick Start

```bash
# Clone this repository
git clone https://github.com/ives22/mcp-tools.git
cd mcp-tools

# Explore available tools
ls servers/
ls tools/
```

## 📦 Adding a New Tool

1. Create your tool under the appropriate directory (`servers/` or `tools/`)
2. Include a `README.md` with setup instructions
3. Provide an `mcp.json` configuration if applicable
4. Submit a pull request
